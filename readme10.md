# Day 10 – First NLP Pipeline

## Overview
This project demonstrates a basic NLP pipeline using Python.

## Pipeline Steps
- Text cleaning
- Lowercase normalization
- Punctuation removal
- Tokenization
- Stopword removal
- Sentiment analysis

## Engineering Observations
Raw text contains inconsistencies and noise.

Preprocessing makes data easier for NLP systems to understand.

Tokenization and filtering improve relevance.

Sentiment analysis extracts emotional meaning from text.

## Tools Used
- Python
- Google Colab
- NLTK
- TextBlob
